title: Java  
description: "Apa itu Java? Java adalah bahasa pemrograman tingkat tinggi yang dikembangkan oleh Sun Microsystems, yang sekarang dimiliki oleh Oracle."  
date: Dec 9 2024  
---  
Java banyak digunakan dalam pengembangan aplikasi desktop, web, dan mobile. Kelebihan Java adalah kemampuannya untuk dijalankan di berbagai platform tanpa perubahan kode, berkat prinsip 'Write Once, Run Anywhere' (WORA).
