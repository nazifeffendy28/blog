---
title: http://  
description: "Apa itu http://? HTTP (HyperText Transfer Protocol) adalah protokol yang digunakan untuk mentransfer data di web, memungkinkan komunikasi antara browser dan server."  
date: Dec 15 2024  
---  
HTTP digunakan untuk mengakses dan mengirimkan data antar server dan client (browser). Ketika Anda mengunjungi situs web, seperti http://www.example.com, browser Anda menggunakan HTTP untuk meminta dan menerima halaman web dari server.
